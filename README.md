# Interactive Sales Dashboard

## Internship Project - Cognetix Technology

This project analyzes superstore sales data and develops an interactive sales dashboard using Python, Pandas, Plotly, Matplotlib, and Seaborn.

The dashboard helps visualize sales performance, profit trends, regional revenue, and top-performing products through interactive charts and business insights.

---

## Objective

The main objective of this project is to:
- Analyze sales performance across regions and product categories
- Study revenue and profit trends
- Identify top-selling products
- Create interactive visualizations and dashboards
- Generate business insights from sales data

---

## Technologies Used

- Pandas
